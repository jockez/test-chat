# test-chat
A simple chat application with NodeJS and SocketIO

# Disclaimer
The code is not production ready as this was my first time getting my feet wet with Node and Socket.IO.

# Versions
NPM: 6.4.1

NodeJS: v10.15.1

Express: 4.16.4

Socket.IO: 2.2.0


# Running the app
`npm install`

`node server.js`

Runs on localhost:3000
